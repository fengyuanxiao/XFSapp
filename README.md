**Cnpm install 项目依赖的过程中出现Package require os(darwin) not compatible with your platform(win32)的警告问题**

运行一下命令：cnpm rebuild node-sass   然后：cnpm install

以上cnpm可以写成npm，cnpm只是使用了淘宝镜像，这样安装会快一些。
